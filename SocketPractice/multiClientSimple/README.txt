##########################################################
########### Terminal Server #############################

1.terminal Server creating socket and listening on port 8080.
2.Use ctrl + C to terminate the server.
3.Server provide message after receiving client message and accept command-
 	Add A,B  / Sub A,B / Mul A,B / Div A,B 
 and send calculated output to client.


4.Build command - 
	gcc terminalServer.c -o server
